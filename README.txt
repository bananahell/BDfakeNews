Banco de dados de fake news da aula de Banco de Dados UnB 2018/2

Para o banco de dados desse trabalho, deve-se executar, em Python 3, o arquivo
SCRIPTS.py havendo um usuário admin com a senha 123 em localhost no postgres com
um banco de dados chamado Projeto. Então para rodar o funcionamento da interface
interativa, deve-se rodar em Python 3 o arquivo Projeto.py.

O projeto está no repositório https://github.com/bananahell/BDfakeNews
